Project

Resume Builder

The Resume Builder project is a web-based application designed to help users create, customize, and manage their resumes efficiently. It provides a user-friendly interface where individuals can input their personal information, work experience, education, and skills to generate a professional resume. The generated resumes can be downloaded in various formats like PDF or DOCX.

Features:

User Registration and Authentication: Secure sign-up and login features to manage multiple resumes and track user history.
Resume Templates: A variety of customizable templates to suit different professions and preferences.
Data Management: Forms to input and update personal information, work experience, education, and skills.
Preview and Export: Ability to preview the resume in real-time and export it in different formats.
Storage: Saving resumes and user data in a MySQL database for easy retrieval and management.
Languages and Technologies Used:

  • HTML
  • CSS
  • JAVASCRIPT
  • PHP
  • MYSQL
  • APACHE
